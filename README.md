# Guile Perceptron

A basic attempt to create a [Perceptron](https://en.wikipedia.org/wiki/Perceptron) in guile, that can distiguish between circles and squares.

## Getting Started

```console
$ ./perceptron.scm
```

## Requirements

- [Raylib](https://www.raylib.com)
- [guile-raylib](https://github.com/thewalkingforest/raylib-guile)
